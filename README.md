# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will churn using machine learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Algorithm
- Logistic Regression

## Steps Performed
1. Data loading
2. Data exploration
3. Data preprocessing
4. Feature encoding
5. Train-test split
6. Model training
7. Model evaluation
8. Prediction

## Dataset
Public Telco Customer Churn dataset (Kaggle)

## Outcome
The model predicts whether a customer is likely to churn or not.
